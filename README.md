# AI-Powered Incident Deduplication System

An intelligent, real-time industrial incident logging and duplication detection system. Utilizing vector space embeddings via Ollama to compute semantic similarities across streaming incident records, filtering redundancy before database serialization, and rendering states dynamically inside a responsive administrative command console.

## 🚀 Key Features
* **AI-Driven Interception:** Processes textual anomalies via local embedding pipelines and models (`TinyLlama`) to cluster matches safely.
* **Operational Control Room:** Real-time dashboard view with high-fidelity analytics cards tracking deduplication metrics.
* **Status Lifecycle Controller:** Granular management permissions to resolve, close, or audit tracking entries interactively.
* **Self-Contained Deployment:** Simple root entry execution bridging frontend interfaces with localized API engines effortlessly.

## 🛠️ Tech Stack
* **Backend:** Flask (Python)
* **AI Processing:** Ollama Embeddings & RAG Framework
* **Frontend:** Tailwind CSS, JavaScript (Asynchronous Fetch API), FontAwesome
* **Data Layer:** SQLite / Local Memory State Mapping
